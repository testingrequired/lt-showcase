## Installation

1. Clone this repo
2. `$ nvm use` or have at least Node LTS installed
3. `$ npm i`

## Build

- `$ npm run build`

## Tests

- `$ npm test`

## Run

### Development

- `$ npm start`

### Production

1. `$ npm run build`
2. `$ npx serve build`
